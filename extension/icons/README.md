# ROSETTA Extension Icons

## Required Icons

The extension needs the following icon sizes:

- **icon16.png** (16x16) - Toolbar icon
- **icon48.png** (48x48) - Extension management page
- **icon128.png** (128x128) - Chrome Web Store

## Creating Icons

### Option 1: Use Online Tools
Visit https://favicon.io/favicon-generator/ or similar to create:
- Simple "R" letter icon with gradient purple background
- Download as PNG in required sizes

### Option 2: Use Design Software
1. Create 128x128 PNG with purple gradient background (#667eea to #764ba2)
2. Add white "R" or "ROSETTA" text
3. Export as PNG
4. Resize to 48x48 and 16x16

### Option 3: Quick Placeholder (Temporary)
Use any square PNG image and rename to:
- icon16.png
- icon48.png
- icon128.png

## Color Scheme

Match the extension's gradient:
- Primary: #667eea (Purple)
- Secondary: #764ba2 (Darker purple)
- Text: White (#FFFFFF)

## Temporary Solution

For testing, you can use simple colored squares:
1. Create 128x128 purple square PNG
2. Duplicate and resize to 48x48 and 16x16
3. Place in `extension/icons/` folder

The extension will work without icons but Chrome will show a default placeholder.
